# 看板娘素材索引 (Mascot Assets Index)

本文档用于管理网站看板娘（Mascot）的图片素材、命名规范及应用场景。

## 📂 目录位置
`src/assets/images/mascot/`

## ⚠️ 命名规范
为了确保在不同操作系统（Windows/Mac/Linux）和构建工具（Vite/Git）中的兼容性，**强烈建议使用小写英文文件名**，单词之间用连字符 `-` 或下划线 `_` 分隔。

| 中文描述 | 建议文件名 |
| :--- | :--- |
| 开心 | `happy.png` / `success.png` |
| 疑惑 | `confused.png` / `404.png` |
| 睡觉 | `sleep.png` / `empty.png` |
